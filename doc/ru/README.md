# Документация TidySnap

Добро пожаловать в документацию **TidySnap** — утилиты для macOS с TUI-интерфейсом для автоматической очистки скриншотов и записей экрана.

## Оглавление

- [Установка](installation.md) — сборка из исходников и установка в систему
- [Использование](usage.md) — TUI-режим, CLI-флаги и управление демоном
- [Конфигурация](configuration.md) — формат конфига, пути и параметры
- [Архитектура](architecture.md) — структура проекта и описание модулей
- [Безопасность](security.md) — защитные механизмы и ограничения
- [Разработка](development.md) — сборка, тесты и вклад в проект
