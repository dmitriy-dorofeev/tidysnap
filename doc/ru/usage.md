# Использование

## Режимы работы

TidySnap поддерживает два режима работы: **интерактивный TUI** (по умолчанию) и **CLI** с флагами.

## TUI-режим (по умолчанию)

Запуск без аргументов открывает интерактивный интерфейс:

```bash
tidysnap
```

### Первый запуск (Welcome → Setup)

1. **Экран приветствия** — нажмите `Enter` или `s`, чтобы начать настройку.
2. **Выбор папки** — перемещайтесь `↑`/`↓`, входите в папки `→`/`Enter`, возвращайтесь `←`, нажмите `Пробел`, чтобы **выбрать** текущую папку.
3. **Форма настроек**:
   - **Расширения файлов** — через запятую, например: `.png, .mov, .mp4`
   - **Срок хранения (дней)** — файлы старше указанного срока будут удалены
   - **Интервал проверки (часов)** — как часто запускать фоновую очистку
   - **Тестовый режим (Dry Run)** — показывать файлы, но не удалять
4. **Предупреждение** — если выбрана одна из системных папок (`Desktop`, `Downloads`, `Documents`, `Movies`, `Music`, `Pictures`, `Public`, `Library`), появится предупреждение.
5. **Сохранение** — настройки сохраняются в `config.yaml`, и вы переходите на экран статуса.

### Экран статуса (Status)

Отображает текущие настройки и состояние демона:

| Клавиша | Действие |
|---------|----------|
| `r` | Запустить очистку (сканирование + предпросмотр) |
| `l` | Открыть логи |
| `e` | Изменить настройки (выбор папки и параметров) |
| `s` | Действие с демоном (зависит от состояния): Установить → Загрузить → Запустить → Остановить |
| `x` | Удалить настройки и plist (сброс) |
| `q` / `Esc` | Выход |

### Экран предпросмотра (Preview)

После сканирования (`r`) показывается список файлов, которые будут удалены. Если включён **Dry Run**, файлы реально не удаляются — только логируются.

| Клавиша | Действие |
|---------|----------|
| `d` / `Enter` | Выполнить очистку (удаление или dry-run) |
| `q` / `Esc` | Назад к статусу |

## CLI-флаги

| Флаг | Описание |
|------|----------|
| `--cleanup` | Запустить фоновую очистку (без TUI, для `launchd`) |
| `--config-path` | Показать полный путь к файлу конфигурации |
| `--reset` | Сбросить настройки (удалить `config.yaml`) |
| `--uninstall` | Удалить plist, конфиг и выгрузить демон |
| `--version` | Показать версию, коммит и дату сборки |

### Примеры

```bash
# Показать путь к конфигу
tidysnap --config-path

# Сбросить настройки
tidysnap --reset

# Полное удаление
tidysnap --uninstall
```

## Управление демоном (launchd)

TidySnap использует `launchd` для фоновой работы. Управление происходит через TUI (клавиша `s`), но можно управлять и вручную:

```bash
# Загрузить демон
launchctl load ~/Library/LaunchAgents/com.tidysnap.plist

# Выгрузить демон
launchctl unload ~/Library/LaunchAgents/com.tidysnap.plist

# Запустить немедленно
launchctl start com.tidysnap

# Остановить
launchctl stop com.tidysnap

# Проверить статус
launchctl list | grep com.tidysnap
```

Демон запускает команду `tidysnap --cleanup` с заданным интервалом (`StartInterval` в секундах).
