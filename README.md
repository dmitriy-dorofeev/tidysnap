# TidySnap

Утилита для macOS с TUI-интерфейсом для автоматической очистки скриншотов и записей экрана.

## Возможности

- 🔍 Поиск файлов **по расширениям** (не по имени) — работает с любыми языками системы
- ⚙️ Настройка папки, расширений и срока хранения через интерактивный TUI
- ⚠️ Предупреждение при выборе системных папок (`Desktop`, `Downloads`, `Documents`)
- 🧪 Тестовый режим (Dry Run) по умолчанию — сначала показывает, что будет удалено
- 📜 Логирование всех операций
- 🔄 Фоновая работа через `launchd`

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/dmitriy-dorofeev/tidysnap.git
cd tidysnap

# Собрать и установить
make install
```

## Использование

```bash
# TUI режим (по умолчанию)
tidysnap

# Фоновая очистка (для launchd)
tidysnap --cleanup

# Показать путь к конфигу
tidysnap --config-path

# Сбросить настройки
tidysnap --reset

# Удалить plist и конфиг
tidysnap --uninstall

# Показать версию
tidysnap --version
```

## Пути на macOS

| Ресурс | Путь                                                 |
|--------|------------------------------------------------------|
| Конфиг | `~/Library/Application Support/tidysnap/config.yaml` |
| Логи   | `~/Library/Logs/tidysnap/`                           |
| Plist  | `~/Library/LaunchAgents/com.tidysnap.plist`          |

## Безопасность

- Первый запуск работает в тестовом режиме (Dry Run)
- Поиск только по расширениям, не по имени файла
- Запрет на выбор корневых системных папок
- Предупреждение перед работой с `Desktop` / `Downloads` / `Documents`
- Все удаления логируются
